# tavpblocks

Library **komponen UI** untuk TAVP, dibangun dengan Tailwind + Alpine + Volt.
Target: 40+ komponen jadi yang tinggal dipakai.

## Komponen (rencana)

Tombol, input, select, modal, dropdown, toast, card, badge, avatar,
datatable, pagination, form-group, alert, skeleton, empty-state, tabs,
accordion, file-upload, rich-editor, date-picker, chart, stat-card, dll.

## Cara pakai (rencana)

```
tavp module:install tavp/tavpblocks
```

Lalu panggil komponennya langsung di template Volt.

## Status

Planning. 15 komponen dasar di `0.2.0`, lengkap 40+ di `0.4.0`.
